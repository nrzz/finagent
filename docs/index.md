# FinAgent documentation

Self-hosted AI finance agent for stocks, mutual funds, F&O, and crypto.

Start with [How to use](../HOW-TO-USE.md) or [Installation](guides/installation.md).

- **Phone APK:** [Android](android.md) — download from GitHub Releases (no Android Studio)
- **Publish a release:** [Release guide](guides/release.md)

**Not financial advice.** Paper trading is the default. Live trading requires explicit configuration and confirmation.
